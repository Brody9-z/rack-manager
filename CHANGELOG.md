# 📋 更新日志

---

## v0.9 — 2026-07-08

### 新增
- 📡 新增独立天线放置功能（不在机柜内，放在场景中任意位置）
- 📡 工具栏「放置天线」按钮，点击场景地面即可放置
- 📡 3D室外单元天线模型：三脚架 + 主机箱 + 馈源发射头 + 信号波纹
- 🖱️ 天线支持移动模式拖拽 + 右键菜单编辑/删除
- 📋 侧面板显示天线列表及位置信息

### 变更
- 天线从机柜设备类型中移除，改为独立放置（更符合实际场景）
- 演示数据包含一个示例天线

### 优化
- 移动模式现在支持同时拖拽机柜和天线

## v0.8 — 2026-07-08

### 新增
- 🗑️ 删除机柜功能（工具栏按钮 + 侧面板 + 右键菜单）
- 🖱️ 3D场景右键菜单（右键机柜/设备/连线可操作删除或编辑）
- 🔗 右键连线可直接删除
- 📜 CHANGELOG 更新日志文件

### 优化
- 删除机柜时自动清理关联的所有设备与连线
- 删除设备时自动清理相关连线
